# The notebook
__Записная книжка.__
*Требования: класс Contact с полями name, phone, email; класс PhoneBook с методами add, remove, findByName, findByPhone; хранение контактов в коллекции; сохранение/загрузка всех контактов в/из текстового файла (формат на выбор студента — например, CSV)*

Это учебное приложение представляет собой электронную записную книжку, которая работает в консоли. Программа позволяет добавлять контакты с именем, телефоном и email, искать их по имени или номеру телефона, удалять ненужные и сохранять все данные в текстовый файл. При следующем запуске контакты автоматически загружаются обратно.
